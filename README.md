Nombre del proyecto: dCore

Descripción: 
Este proyecto es una página web básica desarrollada como parte del curso Front-End JS brindado por Talento Tech. La página está estructurada con HTML semántico y utiliza las etiquetas header, main y footer para organizar el contenido.

El objetivo de esta página web es transformarla en un espacio de ideas originales. Mi propuesta es bastante sencilla: subir comics, e-books y más adelante audiolibros para compartirlos con una audiencia.

En una futura versión:

Los usuarios ademas de consumir estas obras también podrán optar por subir las propias. Este proceso sería evaluado más adealante por una IA para evitar plagios y dará recomendaciones para modificarlos o luz verde para la publicación.
